How to use software installed as a Module:

The environment management package Module is essential for running software on most PSC systems.

To list all of the installed software and their versions, type in:

$ module avail

To load the environment for a software package, which defines paths and variables needed to use the software, type in:

$ module load software-name

For example, to load the fiji software package:

module load fiji

To view available help for the module type in:

module help software-name.

For example:

module help fiji

How to use interactive visualization software:

Some software (fiji/Vaa3d/rstudio and others) have graphical user interfaces. To use software that contains a gui, the X2Go software client should be installed locally, then used to connect to the BIL system. Here is how you would do that:

  • Download the X2GO clientfor your machine from: https://wiki.x2go.org/doku.php
  • Start the X2Go client software on your machine.
  • From the SESSION menu select NEWSESSION.
  • In the “Session preferences” Popup:
    • Set the Session Name to: viz.brainimagelibrary.org
    • Under Server, set the Host to: viz.brainimagelibrary.org
    • Under session type, set the selection to: MATE
    • Click the [OK] Button
  • Move the mouse cursor to the light gray box over the words: “viz.brainimagelibrary.org
  • click the LEFT mouse button
  • Enter your (login) username and your password then click the [OK] Button
  • After a few seconds, an X2GO desktop will appear on your screen
  • When you are done, be sure to select shutdown to completely terminate the X2Go session.

How to use specific software packages:

  1. Anaconda: Anaconda is installed as a module To run Anaconda:

    $ module load anaconda $ python3

    After loading the module, users can also run:
    • ipython
    • Jupyter Notebooks
    • Jupyter Labs
    Additional packages that are included in anaconda include: allensdk, biopython, bokeh, Dask, napari, nilearni, neuron, pandas, pytorch, scikit-image, scikit-learn, starfish, Theano. To see a full list of all available packages, type in terminal

    $ pip list

  2. imagej-fiji: Fiji is installed as a module
    3. To run imagej/fiji:

    $ module load fiji $ fiji

  • julia: Julia is installed as a module
    • To run Julia:

    $ module load julia $ julia

  • knime: Knime is installed as a module
    • To run Knime:

    $ module load knime $ knime

  • matlab: Matlab is installed as a module
    • To run Matlab, users first need to fill out the following form: https://www.psc.edu/resources/software/matlab/permission-form/ After permission is granted,

    $ module load matlab $ matlab -options < input-file > output-file

    After loading the module, to see what toolkits are available:

    $ ver

    Additional documentation: http://www.mathworks.com/products/matlab/
  • R and Rstudio:is installed as a module
    • Interactive use of R and RStudio are available through Anaconda:

    $ module load anaconda $ conda activate rstudio

  • scala: Scala is installed as a module
    • To run Julia:

    $ module load julia $ julia

  • vaa3d: Vaa3d is installed as a module
    • To run Julia:

    $ module load julia $ julia

    Use of Pre-publication Data

    The Library expects all users of the submitted data, unless explicitly granted by the data submitter or the data submitter's project, to grant the authors of the data the right to publish the first paper concerning the dataset within three years of deposit. If there is no publication associated with the dataset, please contact the listed authors for permission to publish. There is no restriction on post-publication data.
