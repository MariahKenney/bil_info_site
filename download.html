Downloading Data

Overview

There are many supported ways to download datasets from BIL including anonymous ftp, and Globus Online. This section discusses how to locate datasets in BIL and download them using these methods.
  1. Locating Data: Datasets in BIL are uniquely referred to by a combination of a 16 digit unique identifier and the dataset name separated by a period.
  2. Anonymous FTP: How to download datasets using anonymous FTP.
    1. Using a web browser: How to download datasets using anonymous FTP through a web browser.
    2. Using wget: How to download datasets using unix wget command.
    3. Using other clients:Advice on how to download datasets using other ftp clients.
  3. Globus: How to download datasets using Globus.

Download Steps

  1. Locating Data

    Datasets in BIL are uniquely referred to by a combination of a 16 digit unique identifier and the dataset name separated by a period:

    <16 digit unique identifier>.<Dataset Name>

    for example: 21f897a23b2674c1.AW326532-01

    The file path to BIL entries will be:
    /bil/data/<c1c2>/<c3c4>/<16 digit unique identifier>/<Dataset Name>.</

    where <c1c2> are replaced by the first and second characters of the unique identifier and <c3c4> are replaced by the third and fourth characters of the unique identifier. for example, the dataset referred to above (21f897a23b2674c1.AW326532-01) would be found down the following path:

    /bil/data/12/f8/12f897a23b2674c1/AW326532-01/

Anonymous FTP - Using a web browser

To access the files, enter the following URL in your web browser ftp://download.brainimagelibrary.org:8811/. Note: most web-browser anonymous ftp clients will place you in the /bil/data directory as the root directory.

Anonymous FTP - wget client

To access the files, enter the complete URL to the file in your web browser, without the /bil/data path ftp://download.brainimagelibrary.org:8811/. wget ftp://download.brainimagelibrary.org:8811/00/00/README

Anonymous FTP - Command-line client

To access the files, enter the following URL in your web browser ftp://download.brainimagelibrary.org:8811/. Note: most web-browser anonymous ftp clients will place you in the /bil/data directory as the root directory.

ftp://download.brainimagelibrary.org:8811/

Image file download formats

Image files can be downloaded in the format as submitted to the library. Additional download file formats are expected to be added in 2018.

Metadata file download formats

Metadata for each dataset will be downloaded in either: json, xml, or tab delimited formats.

Embargoed entries

Data submitted to the library, by default, will be released as soon as possible once data has been validated. Optionally, data submitters can select a limited embargo period for their submitted data of up to one year. Data subjected to embargo can not be downloaded.

Citation of entries and acceptable use

Use of Pre-publication Data. The Library expects all users to grant the authors/creators of the data the right to publish the first paper concerning the dataset within three years of deposit into BIL. If there is no publication associated with the dataset, please contact the listed authors for permission to publish.
