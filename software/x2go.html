Software Available on BIL Systems: X2Go

Overview

X2Go is an open source remote desktop application that is installed on all BIL VM's. X2Go enables one to connect to and run traditional desktop visualization applications (such as Fiji - ImageJ) on a remote computing system.

This document outlines how to use the X2GO remote desktop application to access a virtual machine at BIL.

To use software that contains a gui, the X2Go software client should be installed locally, then used to connect to the BIL system.

If you have problems using the X2go software, do not hesitate to contact the BIL Helpdesk for assistance.

  2. Connecting to a BIL VM (workshop.brainimagelibrary.org) using X2Go

Install X2Go client

Download and install the appropriate x2go client (Windows, Linux, or Mac) from: https://wiki.x2go.org/doku.php

Mac Users: You must also install xQuartz in order to run x2go: https://www.xquartz.org

Connecting to a BIL VM (workshop.brainimagelibrary.org) using X2Go

Below is an example using X2GO to connect to the BIL VM workshop.brainimagelibrary.org If you are connecting to a different BIL VM, substitute the name of that VM for workshop.brainimagelibrary.org in the instructions below.

  • Start the X2Go client software on your machine.
  • From the Session menu select New Session.
  • In the Session preferences Popup:
    • Set the Session Name to: workshop.brainimagelibrary.org
    • Under Server, set the Host to: workshop.brainimagelibrary.org
    • Under session type, set the selection to: MATE
    • Click the [OK] Button
  • On the Right side of the screen, you should see a grey box with the session information you entered:
  • Move the mouse cursor to the light gray box over the words: workshop.brainimagelibrary.org
  • Click the LEFT mouse button
  • Enter your (login) username and your password then click the [OK] Button
  • After a few seconds, an X2GO desktop will appear on your screen
  • Open up a terminal, then run any program that produces graphical output.
  • When you are done, be sure to select the System menu then select Shut Down to completely terminate the X2Go session.
